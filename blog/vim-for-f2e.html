为前端工程师打造的 Vim 设置

  • CSS one-line / multi-line folding : Folded text is compressed on one line 前端工程师在书写 CSS 时，一般有两种风格： 横排书写，多个 CSS 属性都写为一行，这样可以节约可视空间，一屏内可以看到更多的代码。 竖排书写，每个 CSS 属性占一行，这样在编辑 CSS 属性时更方便些，调整各个属性的顺序甚至可以使用 :sort 命令。 这个插件则是将竖排书写的 CSS 属性折叠显示在一行。 我自己也有一些补充，有空处理一下一并奉上。
  • JavaScript Indent : Javascript indenter (HTML indent is included) 2 Javascript(包含 HTML 中的内联 Javascript)代码自动缩进的增强，已经很好了，虽然还有提升空间。
  • JsLint Helper : a javascript code validator - 使用 jslint 验证脚本，并通过 Vim 的 sign 指示错误行。
